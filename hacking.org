* 序列图
** 设计序列图定义语言
 - 字符串定义一个<node> 
   - [*][type.]<name>
 - 符号定义一个消息<message>
   - ‘:=>’<node> 同步消息
   - ‘:->’<node> 异步消息
  
- :=>'<node>/<return> 同步消息带返回值
   - ':#>'<node> 空消息

 - 定义一个消息组<group>
   - '(' progn  { <message> } ')'
   - '(' chain  { <message> } '}'
 - 定义框架消息
   - '(' opt <guard> <message> ')'
   - '(' loop <guard> <message> ')' 
   - '(' if <guard> <message>  <message> ')'
